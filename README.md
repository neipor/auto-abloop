# Auto A-B Loop Player (自动 A-B 循环播放器)

这是一个用 Rust 编写的应用程序，旨在解决游戏背景音乐（BGM）在非游戏环境下播放时循环体验不佳的问题。许多游戏的 BGM 在游戏中是无缝循环的，但发布到 OST（原声带）时，通常会以“开头 -> 循环一两次 -> 淡出”的形式呈现，这在听感上具有很强的割裂感。

`Auto A-B Loop Player` 能够智能检测音频文件中的循环点，并允许用户以无缝循环的方式播放或导出这些音乐，重现如同在游戏中一般的听觉体验。

## 特性 (Features)

*   **智能循环检测**: 自动识别音频文件中的循环起始点和结束点，尤其针对带有淡出效果的游戏 BGM 进行优化。
*   **多格式支持**: 支持 FLAC, MP3, M4A 等多种音频格式，甚至可以从 MKV 等视频文件中提取音轨进行处理。
*   **GUI 和 CLI 模式**:
    *   **图形用户界面 (GUI)**: 提供一个直观的音频播放器界面，支持文件拖拽、音量控制、循环次数设定，并显示专辑封面、歌曲信息及循环检测结果。
    *   **命令行界面 (CLI)**: 允许通过命令行指定文件播放或导出，灵活控制循环次数。
*   **无限循环播放**: GUI 和 CLI 默认都支持无限循环播放模式。
*   **循环导出**: 可以将检测到的循环片段按指定次数导出为 WAV 格式文件。
*   **多语言支持**: 支持中文和英文界面切换。
*   **跨平台**: 支持桌面版 (Windows, macOS, Linux) 和 Web (WASM) 平台。
